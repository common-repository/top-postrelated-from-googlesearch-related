== Installation ==

This section describes how to install the plugin and get it working.

1. Delete any existing `toppostrelated` folder from the `/wp-content/plugins/` directory
2. Upload `toppostrelated` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the options panel under the 'Settings' menu and add your GOOGLE API-KEY
5. Create a new post and get the suggestion or posterelated from GoogleSearch
